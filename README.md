# Excel-end-to-end-project

📊 Superstore Sales Dashboard

This project presents an interactive Superstore Sales Dashboard designed to analyze and visualize sales performance across different regions, categories, and time periods. The dashboard provides meaningful insights that help in understanding business trends and making data-driven decisions.

🚀 Features
Sales Overview: Displays total sales, average sales, total transactions, and total quantity.

Region-wise Analysis: Compare performance across Central, East, South, and West regions.

Ship Mode Distribution: Visual breakdown of sales by shipping modes (First Class, Second Class, Same Day, Standard Class).

Monthly Trends: Line chart showing quantity ordered month-wise.

Top Cities: Highlights the top 5 cities based on sales.

Category-wise Transactions: Insights into Technology, Furniture, and Office Supplies.

Consumer Segmentation: Quantity distribution across Consumer, Corporate, and Home Office segments.

Year & Region Filters: Interactive filters to dynamically explore the data.


📈 Purpose

The main goal of this project is to:

Analyze sales performance efficiently

Identify high-performing regions and categories

Understand customer purchasing behavior

Support better business decision-making
